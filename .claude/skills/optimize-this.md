# 代码优化

优化用户当前打开或选中的代码：

1. **分析当前代码**
   - 读取并理解代码逻辑
   - 识别性能瓶颈
   - 发现代码问题（冗余、低效等）

2. **提出优化方案**
   - 性能优化（减少计算、缓存、算法改进）
   - 代码质量（可读性、可维护性）
   - 内存优化（减少 GC、对象池等）

3. **针对性优化**
   - Shader：减少指令、优化采样
   - C# 脚本：优化 Update、减少装箱、使用 Job System
   - 渲染：批处理、LOD、剔除优化

4. **实施优化**
   - 询问用户是否应用优化
   - 修改代码并解释改动
   - 保持功能不变

注意：优化要实用，不要过度工程化。
